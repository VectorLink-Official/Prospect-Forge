THE PROBLEM

記録は取っている。
でも、活かせていない。

フィジカルデータは報告をして終わりか、紙のノートやLINE、パソコンに散らばったまま。せっかくの記録をどう活かしていくかが分からない。

BEFORE
  • 記録用紙とLINEがバラバラで、誰がいつ何を報告したか追えない
  • 個人の成長やチーム内の立ち位置が、数字として見えない
  • 停滞している選手に気づくのが、いつも遅れる
  • 声かけや個別連絡が、すべて手作業
→
AFTER
  • 今日の報告状況がダッシュボードに一覧表示される
  • 総合指数とランキングで、立ち位置が一目でわかる
  • 停滞日数を自動検知し、コーチに知らせる
  • 個別・一斉メッセージをアプリ内のスレッドで完結
FEATURES

必要な機能だけを、ひとつに。

現場で実際に使われながら磨き上げてきた、過不足のない機能セットです。

01 / SCORE

総合指数 & ランク判定

ベンチ・スクワット・デッドリフトの10RMと懸垂回数から、独自の重み付け計算で「総合指数」を自動算出。S〜Dの5段階でフィジカルレベルを可視化します。

02 / VBT

VBTレポート

重量・速度・回数・セット数をセットごとに記録できるVBT(速度ベースドトレーニング)対応の入力フォーム。種目は動的に追加可能です。※フルカスタムプラン

03 / LOG

自由種目ログ

規定種目以外のトレーニングも、種目名・重量・回数を自由記述で記録。補強種目や自主練の積み上げも見逃しません。※フルカスタムプラン

04 / RANK

ランキング

当日の報告ベース、歴代ベストベースの両方でチーム内ランキングを表示。グループ単位での絞り込みにも対応します。※フルカスタムプラン

05 / GROUP

グループ管理

学年やポジション、レベル別にグループを作成し、グループごとに異なる目標値・停滞判定日数を設定できます。※フルカスタムプラン

06 / TALK

スレッド型メッセージ

選手⇄コーチの個別連絡はもちろん、複数選手への一斉送信、グループ単位での一括送信にも対応したメッセージ機能。
